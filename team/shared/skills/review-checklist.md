# 共享技能: 代码审查清单

## 快速审查（5 分钟）
- [ ] 代码能跑吗？有语法错误吗？
- [ ] 有硬编码的密钥/密码吗？
- [ ] 有明显的安全漏洞吗？
- [ ] 函数名/变量名清晰吗？

## 标准审查（15 分钟）
以上 +
- [ ] 错误处理完整吗？
- [ ] 输入验证了吗？
- [ ] 有测试吗？覆盖率够吗？
- [ ] 代码风格一致吗？
- [ ] 文件大小合理吗？（< 800 行）
- [ ] 函数大小合理吗？（< 50 行）
- [ ] 有不必要的复杂度吗？

## 深度审查（30 分钟）
以上 +
- [ ] 架构合理吗？符合项目模式吗？
- [ ] 性能可接受吗？有 N+1 查询吗？
- [ ] 并发安全吗？有竞态条件吗？
- [ ] 边界条件处理了吗？
- [ ] 向后兼容吗？
- [ ] 文档需要更新吗？

## 审查结果模板
```
总评: [PASS / PASS_WITH_COMMENTS / NEEDS_WORK / REJECT]

CRITICAL:
- 文件:行号 - 问题描述 - 修复建议

HIGH:
- 文件:行号 - 问题描述 - 修复建议

SUGGESTION:
- 建议内容

HIGHLIGHTS:
- 做得好的地方
```
