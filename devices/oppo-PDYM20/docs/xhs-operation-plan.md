## AI生成内容标签策略

小红书要求标注AI生成内容。我们的策略：

1. **图片**: Gemini 生成 → 发布时在「高级选项」中标注「图片由AI生成」
2. **文案**: DeepSeek 生成初稿 → 人工（或Claude）润色 → 标注「AI辅助创作」
3. **态度**: 大方标注，反而成为卖点 — "我就是用AI做的，而且效果很好"
4. **内容本身就是AI主题**: 标注AI生成不矛盾，反而增加可信度

### 发布时操作步骤
```
高级选项 → 声明 → 内容包含AI生成
```
